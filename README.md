# Student Grade Tracker App

A Flutter app for students to track subjects, marks, and grades.

## Features

- Add subjects with name and mark (0-100)
- View all subjects with grades
- Swipe to delete subjects
- Summary with total subjects, average mark, and overall grade
- Light/Dark theme toggle
